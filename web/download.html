Source Code

TGZ

  • Working source code is released at the same time as every working binary, through the normal SourceForge downloads page.
  • Daily development source code tarballs are available, although there is no guarantee these will compile, no less run. These are generally under 20MB in size.
  • Developers are advised to start using the CVS updates service if they want regular copies of working source code without having to download the entire source tarball each time (see below).

CVS

The source code is available through SourceForge CVS, and it can be checked out through anonymous (pserver) CVS with the following instruction set. When prompted for a password for anonymous, simply press the Enter key.

cvs -d:pserver:anonymous@cvs.mobius.sourceforge.net:/cvsroot/mobius login
 
cvs -z3 -d:pserver:anonymous@cvs.mobius.sourceforge.net:/cvsroot/mobius co mobius

Updates from within the source code directory do not need the -d parameter.
