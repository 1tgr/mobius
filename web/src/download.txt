<h2>Source Code</h2>
<h3>TGZ</h3>
<p>You can download the latest version of the MÃ¶bius source as one big tarball 
(about 7MB):</p>
<p><a href="http://cvs.sourceforge.net/cvstarballs/mobius-cvsroot.tar.gz">http://cvs.sourceforge.net/cvstarballs/mobius-cvsroot.tar.gz</a></p>
<p>This is a copy of the latest modifications made to the CVS tree on 
SourceForge. If you just want to download the source and look around then 
download this. If you want to keep your local copy of the source up-to-date (and 
you don't mind setting up the <code>cvs</code> program) you can download the
full tree via the SourceForge CVS interface.</p>
<h3>CVS</h3>
<p>The source code is available through SourceForge CVS, and it can be checked 
out through anonymous (pserver) CVS with the following instruction set. When 
prompted for a password for <i>anonymous</i>, simply press the Enter key.
<p><tt>cvs -d:pserver:anonymous@cvs.mobius.sourceforge.net:/cvsroot/mobius login<br>
&nbsp;<br>
cvs -z3 -d:pserver:anonymous@cvs.mobius.sourceforge.net:/cvsroot/mobius co 
mobius</tt>
<p>Updates from within the source code directory do not need the -d parameter.</p>
